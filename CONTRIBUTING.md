# Как внести вклад

См. README -> «Как помочь».

Обязательно перед отправкой:
- Удалите/замаскируйте все IP/домены/ключи/персональные данные.
- Не тестируйте в сетях, где это запрещено правилами (работа/учёба/организация).
- Не публикуйте сами конфиги. Допустимы только агрегированные результаты, метаданные и наблюдения.
- Добавляйте минимум контекста (см. шаблон ниже).

Шаблон заметки (скопируйте и заполните)

Status: tested / sketch / hypothesis
Confidence: low / medium / high
Дата: YYYY-MM-DD
Тестов: N
Локация: город / страна
ОС / версия: ...
Протокол: WireGuard / VLESS / ...
Клиент / версия: ...
Сеть: Wi-Fi / LTE / ...
Результаты:
- SUCCESS: % (N)
- FAILURE: % (N) - классы: timeout/handshake-fail/... (гипотезы причин опционально)
- ERROR: % (N)
Заметки: ...

Кладите в: topics/atlas-n/notes/ВАШ_НИК-ДАТА.md

Присылайте через Pull Request или issue.
