# Notes (crowd data)

Сюда складываются отчёты участников в формате «НИК-YYYY-MM-DD.md».

Правила:
- НЕ публикуйте IP/домены/ключи/персональные данные.
- НЕ прикладывайте сами конфиги - только агрегированные результаты и наблюдения.
- Всегда указывайте минимум контекста: дата, локация, ОС, протокол, клиент, сеть.
- Причины отказов пишите как «класс» (timeout/handshake-fail/...) и, если уверены, как «гипотезу».
- Имена файлов: НИК-YYYY-MM-DD.md (например: skladkoles-2026-01-30.md)

Спасибо за вклад!
